## 1. 定位与输出

评分衡量“该银行当期基本面质量”，不是股票买卖评级、估值或目标价。“试行”表示规则仍需使用本行历史和同口径同业持续校准。

先输出一张读者摘要：

| 项目 | 必须回答的问题 |
|---|---|
| 评分发布状态 | 正式分数、暂定区间还是暂不发布？ |
| 基本面总分 | 数值、区间或不发布；怎么理解？ |
| 证据充分度 | 高、较高、中、低还是不足？ |
| 主要加分项 | 哪三项改善最重要？ |
| 主要扣分项 | 哪三项风险最重要？ |
| 最大限制 | 哪些缺失或不可比口径可能改变评分？ |

默认报告再输出：

- 总分或暂定分数区间；
- 六个分项分；
- 相对上一报告期变化；
- 前三项加分和扣分原因；
- 加权数据中A/B/C/D/N/A的比例；
- 同业组与历史区间。

26个子项始终在内部完成计算，但只在用户明确要求“完整评分”“评分明细”“审计底稿”或监管复核时展示其权重、本行趋势、同业位置、原始分、加权贡献、证据充分度和核心依据。默认报告不得为了证明计算过程而铺开26行明细。

未完成历史回填校准前，在评分标题中注明“试行”。证据不足而不发布分数时，不得只填N/A；必须解释原因、已有正负证据和所缺数据。

26个子项是唯一计算输入；六个维度分是各自子项加权后的展示结果。不得给六个维度另行主观打分、再反推子项或总分。

## 2. 评分结构

| 维度 | 权重 | 子项 |
|---|---:|---|
| 收入与利润质量 | 20 | NII（净利息收入）4；手续费4；其他非息质量3；费用效率3；减值/税/一次性项目6 |
| 资产负债结构与NIM（净息差） | 20 | 资产投放5；NIM（净息差）及重定价5；资产收益率4；负债结构/成本4；流动性与期限2 |
| 资产质量与风险流量 | 30 | 前瞻指标6；新生成不良7；风险结构6；处置依赖4；拨备缓冲5；不良认定严格性2 |
| 资本与增长质量 | 15 | CET1（核心一级资本）/RWA（风险加权资产）5；RORWA（风险加权资产收益率）4；内生资本3；分红/再融资3 |
| 客户与轻资本业务 | 10 | AUM（管理客户总资产）/财富管理4；中收与客群3；科技/产品/生态可持续性3 |
| 治理、披露与股东回报 | 5 | 披露质量2；管理层执行1；股东回报和资本纪律2 |

子项权重之和必须为100。

## 3. 0—5级原始分

每个子项先得到0—5的原始分，再按权重换算：

`加权贡献 = 原始分 ÷ 5 × 子项权重`

原始分、权重和加权贡献是三个不同概念。每项必须满足 `0 ≤ 原始分 ≤ 5`、`0 ≤ 加权贡献 ≤ 权重`；总分等于全部加权贡献之和。报告不得把原始分写入“贡献”列，也不得出现“子项得分大于权重”。

### 3.1 同业截面分

在可比同业组内使用百分位，负向指标先反向：

| 百分位 | 分数 |
|---:|---:|
| ≥80% | 5 |
| 60%—80% | 4 |
| 40%—60% | 3 |
| 20%—40% | 2 |
| <20% | 1 |
| 重大安全、诚信或资本约束问题 | 0 |

### 3.2 本行趋势分

使用最近8个可比季度的分布：

| 本期位置/趋势 | 分数 |
|---|---:|
| 显著改善且位于自身历史上分位 | 5 |
| 温和改善或位于历史较好区间 | 4 |
| 基本稳定或位于历史中位 | 3 |
| 温和恶化或位于历史较差区间 | 2 |
| 显著恶化且位于历史下分位 | 1 |
| 结构性失速、数据失真或安全性断点 | 0 |

### 3.3 合成原始分

`原始分 = 趋势分 × 60% + 同业分 × 40%`

只有一个维度可得时，使用该维度分数并将可信度至少降一级。经营模式差异明显的银行不得强行放入同一同业组。

## 4. 证据充分度和暂定区间

读者显示统一使用：

| 内部等级 | 读者显示 | 含义 |
|---|---|---|
| A | 高（A级） | 财报直接披露，口径明确 |
| B | 较高（B级） | 只使用披露值和确定公式 |
| C | 中（C级） | 依赖少量可验证假设 |
| D | 低（D级） | 重要流量缺失，只能做代理估算 |
| N/A | 不足（N/A） | 无法形成可靠判断 |

对A/B级原始分不加额外不确定性；C级为原始分±1级；D级为原始分±2级；N/A为0—5级。上下限限制在0—5。

如果C/D/N/A子项权重之和超过20%，只输出加权分数区间，不输出单一总分。即使不超过20%，也必须披露不确定权重。

如果N/A子项权重超过50%，或计算区间宽度超过40分，区间也不发布；标记“证据不足，暂不发布评分”，并在读者摘要中列出已有正负证据、缺失数据和可能改变判断的条件。内部机械区间可用于质量检查，不得写入一句话结论。

## 5. 跨期和同业校准

- 首次评分前，至少回填本行最近8个季度；不足8期时标记“样本不足”。
- 同业组默认为3—5家，同时匹配银行类型、业务结构和资本计量方法。
- 上一期分数必须用同一版本评分规则回算；不把旧文章主观分数直接并入。
- 评分变化大于5分时，必须列出子项贡献分解；无法解释则停止发布分数。
- 新资本规则、并表范围或会计重述导致不可比时，只比较可比子项。

## 6. 禁止事项

- 不用总分直接推导买入、卖出、目标价或仓位。
- 不为获得某个预期总分而倒推子项分。
- 不在同业口径不一致时做百分位排名。
- 不隐藏N/A、C/D级估算或相反风险信号。
- 不把披露质量低误当成资产质量好。
- 不在默认报告展开完整子项表；需要复核时切换完整评分或审计模式。
