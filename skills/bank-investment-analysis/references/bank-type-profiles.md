# 银行类型侧重点

类型只决定“优先下钻哪里”，不改变完整数据、来源和勾稽要求。先按实际业务结构判断，不只看名称。

| 类型 | 默认优先问题 | 必看细分 |
|---|---|---|
| 国有大行 | 净息差压力能否由规模、低成本存款和非息对冲；资本新规影响 | 对公/零售投放、存款重定价、债券投资、RWA（风险加权资产）、分红 |
| 股份制银行 | 零售风险出清、负债成本和轻资本业务修复是否同步 | 信用卡/消费贷、财富管理、同业负债、核销处置、CET1（核心一级资本） |
| 城商行 | 区域景气、政信/房地产暴露与高增速资本消耗 | 区域/行业集中度、投资资产、对公大户、RWA（风险加权资产）、外部资本 |
| 农商行 | 小微和区域信用成本是否被高息差充分补偿 | 小微/个人经营贷、关注/逾期、拨备、存款稳定性、区域集中度 |
| 零售特色银行 | 客群与财富管理优势能否覆盖零售信用周期 | AUM（管理客户总资产）、信用卡/消费贷/按揭、手续费、获客与资金成本 |
| 对公/交易特色银行 | 低风险对公扩张是否真正提升回报而非只增规模 | 对公行业、票据、交易资产、结算托管、非息波动、RORWA（风险加权资产收益率） |

选择3—5家同业时同时匹配银行类型、资产规模、主要客群、资本计量方法和报告期。若同类样本不足，降低同业结论权重，不用跨类型排名制造精确感。
