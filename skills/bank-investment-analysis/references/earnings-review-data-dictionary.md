## 1. 底稿字段

所有正式财报点评都使用内部 [earnings-review-data-template.csv](../assets/earnings-review-data-template.csv) 检查D001—D144固定核心指标是否覆盖；官方报告存在量价或非息细分时，在内部底稿末尾追加E001起的可选扩展记录。默认点评模式不在报告中展开逐项底稿，只保留关键数据的来源、口径、计算与限制以及模块完整度。用户明确要求审计底稿、逐项追溯、完整数据附录或监管复核时，才把原16个字段合并成一张8列紧凑表追加到报告；同一 `record_id` 只出现一次：

| 字段 | 要求 |
|---|---|
| section | profit/assets/loans/liabilities/risk/provision/capital/score |
| metric | 使用本字典的标准指标名 |
| value | 纯数值；N/A 时留空 |
| unit | 亿元、%、bp、倍、户等 |
| period | 20XXQ1/H1/Q3/FY 或期末日 |
| comparison_period | 上期或上年同期 |
| scope | 集团/母行/子公司 |
| balance_type | 期末/日均/单季/累计 |
| status | disclosed/calculated/estimated/inferred/N/A |
| confidence | A/B/C/D/N/A |
| source_file | 官方报告文件名 |
| page_note | 页码或附注名 |
| formula | 计算/估算公式 |
| assumptions | 估算假设；非估算留空 |
| residual | 勾稽残差 |
| comment | 口径、限制与N/A原因 |

紧凑表映射如下：

| 紧凑列 | 原字段 |
|---|---|
| 编号 | `record_id` |
| 分析模块·指标 | `section`、`metric` |
| 数值·单位 | `value`、`unit` |
| 数据期间·对比期 | `period`、`comparison_period` |
| 范围·统计口径 | `scope`、`balance_type` |
| 数据性质·可靠程度 | `status`、`confidence` |
| 证据文件·页码 | `source_file`、`page_note` |
| 计算方法·假设·限制 | `formula`、`assumptions`、`residual`、`comment` |

审计模式下，组合单元格使用稳定中文标签和 `<br>` 分隔；渲染器提供可逆映射，以便人工阅读和程序恢复。核心 `record_id` 从 `D001` 编到 `D144`；扩展记录从 `E001` 顺序编号，两者均不计入原16个数据字段。内部字段继续使用标准英文键名，读者可见附录不得展示裸露的 `status:`、`confidence:`、`source_file:` 等内部字段。

行业通用英文指标不翻译成自造中文简称；在所有读者可见位置每次都补充中文含义，例如 `QoQ（环比）`、`YoY（同比）`、`NIM（净息差）`。内部字段保留标准键名和原始指标名，审计模式渲染的附录使用可逆中文显示名。

数据性质和可靠程度的读者显示固定为：

| 内部值 | 读者显示 |
|---|---|
| `disclosed` / `A` | 财报直接披露 / 高（A级） |
| `calculated` / `B` | 确定公式计算 / 较高（B级） |
| `estimated` / `C` | 假设估算 / 中（C级） |
| `estimated`或`inferred` / `D` | 假设估算或分析推断 / 低（D级） |
| `N/A` / `N/A` | 未披露或不可得 / 不足（N/A） |

## 2. 报告类型标记

- `A`：年报/中报必选。
- `Q`：一季报/三季报必选。
- `K`：业绩快报最低必选。
- `C`：条件必选；官方披露时必须提取，未披露时填N/A。

## 9. 缺失数据规则

- 必选行不得删除；填N/A并说明是未披露、口径不可比还是无法反推。
- 报告直接披露但未提取，不得标记N/A。
- 季报未披露的行业/区域结构，可引用最近年报/中报作背景，但不冒充本期数据。
- 核心数据若为C/D级，必须在正文披露假设和对结论的影响。

## 10. 可选扩展明细

扩展记录只在官方报告直接披露或能够用确定公式计算时追加，不要求固定数量；未披露时不生成，不批量填N/A。推荐指标如下：

| 模块 | 可选扩展指标 |
|---|---|
| 贷款量价 | 贷款平均余额、贷款利息收入、贷款收益率 |
| 付息负债量价 | 客户存款/同业负债/应付债券/央行借款/计息负债合计的平均余额、利息支出和成本率 |
| 存款细分量价 | 公司活期/定期/小计、零售活期/定期/小计及活期/定期合计的平均余额、利息支出和成本率；目标季度未披露时使用最近完整披露期并标注期间 |
| 手续费 | 财富管理、资产管理、托管、银行卡、结算清算、其他手续费收入、手续费收入合计和手续费支出 |
| 其他非息 | 投资收益、公允价值变动、汇兑、其他业务收入、其他业务成本、OCI（其他综合收益） |
| 费用 | 员工费用、其他业务费用、成本收入比 |

扩展指标必须避免与D核心指标重复。若扩展表中的合计已存在于核心指标，只追加必要分项，在正文和附录中回指核心合计。手续费收入分项属于收入端，不得与手续费净收入直接相加；期末负债余额不得冒充付息负债平均余额。
