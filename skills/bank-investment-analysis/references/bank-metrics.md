# 银行指标与计算

所有金额使用同一单位；利率函数中的输入为百分数。计算可调用 `scripts/bank_metrics.py`。

## 单季重构

`本季金额 = 本期累计金额 − 上一累计期金额`

例如：Q2 = H1 − Q1，Q3 = 9M − H1，Q4 = FY − 9M。

累计平均余额是时间加权量，不能直接相减：

`新增期间平均余额 = (本期累计平均余额×本期月数 − 上期累计平均余额×上期月数) / 新增月数`

调用 `incremental_average_from_cumulative`。金额项通常可精确重构；平均余额、NIM、贷款减值和保险分项应分别标记“精确重构 / 估计重构 / 不可重构”。

同比比较使用本季与上年同季；环比比较使用本季与上季。累计同比不能替代单季趋势。

## 净利息收入与净息差

`净利息收入 = 平均生息资产 × 年化净息差 × 期间月数 / 12`

`反推平均生息资产 = 期间净利息收入年化 / 净息差`

`反推单季净息差 = 单季净利息收入年化 / 单季平均生息资产`

期末生息资产不足时，可用期初期末均值近似，但要标记 `[估计]`。优先使用银行披露日均余额。

披露 NIM 通常只保留两位小数。反推单季时先用 `rounded_value_bounds` 传播 `±0.005个百分点` 的舍入区间；若区间足以改变趋势结论，不报告单一点值。

净息差变化可分解为：

`ΔNIM ≈ Δ资产收益率 − Δ负债成本 + 结构/口径残差`

进一步按贷款、债券、同业资产；存款、同业负债、已发行债券拆解。用重定价批次而不是只用政策利率变动量。

## 实际税率

`实际税率 = (1 − 净利润 / 税前利润) × 100%`

归母净利润与集团税前利润口径可能不完全一致，精确分析优先用净利润。异常低税率通常需检查国债/地方债免税收入、以前年度调整和一次性项目。

## 信用成本

`年化信用成本 ≈ 当期贷款相关减值损失年化 / 平均贷款余额`

若报告只给全部信用减值损失，不要把它无条件当贷款信用成本；债券、同业及表外资产也可能包含在内。

## 新生成不良

不良贷款余额变化及核对：

`期末不良 = 期初不良 + 新生成不良 + 转入/并表 − 现金清收 − 核销 − 批量转让/ABS − 上调正常 − 其他处置 ± 汇率及其他`

运行 `npl_stock_bridge`展示计算期末不良、披露期末不良和残差。残差不为零时列为“未披露净流量”，不修改披露值强行平衡。

未披露新生成不良时，使用：

`新生成不良代理下限 = max(0, 期末不良 − 期初不良 + 已披露或估算的核销/转出 − 已知转入/并表)`

运行 `npl_generation_proxy`计算中性值与区间，并保留函数返回的 `raw_low/raw_high`检查原始公式是否出现负值。该值不得简称为“新生成不良”；只有转入和并表已单独扣除时，忽略现金清收、上调、批量转让等其他处置才构成下限逻辑。汇率或并表影响无法分离时降为D级，并改称“代理值区间”。

方向性结论至少再用关注/逾期/阶段二/迁徙率或细分业务风险中一类信号交叉验证。

## 拨备

- `拨备覆盖率 = 贷款减值准备 / 不良贷款余额`
- `贷款拨备率 = 贷款减值准备 / 贷款总额`

高拨备覆盖率不等于当期资产质量改善；它同时受历史拨备积累、不良认定和核销节奏影响。

### 贷款减值准备变动

`期末贷款减值准备 = 期初准备 + 当期贷款减值计提 + 收回已核销贷款 + 转入/汇率及其他 − 核销/转出`

运行 `provision_rollforward`计算期末准备和残差。

未披露核销/转出时：

`估算核销/转出 = 期初准备 + 贷款减值计提 + 收回已核销贷款 + 其他净流入 − 期末准备`

运行 `estimate_writeoff_range`。贷款减值计提未披露时，使用最近两个完整披露期的“贷款减值/信用减值”构建区间。收回已核销未披露时，使用历史同季和近期节奏构建区间。

并表、汇率、资产包买卖或重大重分类存在时，估算至少降为D级。

## 资产负债结构

- `科目占比 = 科目余额 / 总资产或总负债`
- `占比变化 = 本期占比 − 上期占比`
- `对总增量贡献 = 科目增量 / 总资产或总负债增量`

运行 `balance_sheet_mix`统一计算。总增量为零时，增量贡献标N/A，不用0或无穷大。主要科目合计与总额的差异列为“其他”，不丢在残差。

## 资本与效率

- `RORWA = 净利润年化 / 平均风险加权资产`
- `风险资产生成比 = RWA / 生息资产`
- `新增RWA所需核心资本 ≈ ΔRWA × 目标核心一级资本率`
- `内生资本增量 ≈ 归母净利润 − 普通股分红 − 其他资本扣减`

比较总资产增速与 RWA 增速，识别低权重资产扩张、内评法参数变化和资本套利。评价股东回报时，考虑利润中必须留存以支持 RWA 增长的部分。

资本规则、内评法状态或合并范围变化的过渡期，禁止用不同规则的期初/期末 RWA 计算平均 RWA 或 RORWA。优先使用同规则第三支柱数据、监管重述数或只报告期末资本指标；无法同口径时标记不可比。

## 可信度分级

运行 `confidence_grade`辅助分级：

- A：完整官方报告直接披露；
- B：只依赖A级披露值和确定公式，残差不重大；
- C：依赖少量可验证假设，能给出合理区间；
- D：缺失重要流量，只能给代理下限或宽区间；
- N/A：无可辩护估算。

当残差超过选定参考基数的5%时，计算/估算至少降为D级。如果使用贷款减值作参考基数，可在报告中明确放宽到10%，但必须说明原因。

## 财报评分

使用 [earnings-scorecard.md](earnings-scorecard.md) 确定26个子项的原始分和权重，运行 `fundamental_scorecard`汇总。C/D/N/A加权超过20%时，函数不返回单点总分；只有 `publishable=True` 时才发布 `published_score_range`。N/A超过50%或区间宽度超过40分时，保留内部 `score_range`供检查，对外写“证据不足，暂不发布评分”。

## 利润构成与核对

“三收三支”只用于经营解释。完整法定核对关系为：

`营业收入 = NII + 手续费净收入 + 其他非息`

`税前利润 = 营业收入 − 税金及附加 − 业务及管理费 − 信用减值 − 其他资产减值 − 其他业务成本 + 营业外净额`

`归母净利润 = 税前利润 − 所得税 − 少数股东损益`

少数股东损益可为负。调用 `bank_profit_bridge` 时，费用/损失以正数输入，转回/收回以负数输入；少数股东损益沿利润表符号输入。最后按报告定义处理归属于普通股股东、优先股和永续债影响。

## 成本收入比

不同银行对其他业务成本的处理可能不同。按报告脚注重述口径；并表保险业务时优先使用剔除其他业务成本后的银行可比口径。

## 区间与情景

- 下限 ≤ 中性值 ≤ 上限；
- 净利润对收入、减值、税率的敏感性分别列示；
- 百分比“±2%”与“±2个百分点”不得混写；
- 中性值不必是上下限的算术中点，但偏离必须有明确概率或非线性理由。
